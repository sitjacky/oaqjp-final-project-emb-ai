# Final Project - Emotion Detection Application

This repository contains the **Final Project** for the IBM Developing AI Applications with Python and Flask course.

## Project Name

**Final Project - Emotion Detection Application**

## Description

This project uses the IBM Watson NLP Emotion Prediction service to analyze the emotions expressed in user-provided text. The application is built using Python and Flask and returns emotion scores for:

* Anger
* Disgust
* Fear
* Joy
* Sadness

It also identifies the dominant emotion detected in the text.

## Technologies Used

* Python 3
* Flask
* Requests
* IBM Watson NLP Emotion API
* HTML
* JavaScript
* Git & GitHub
